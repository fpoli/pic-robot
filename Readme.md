PIC Robot
=========

Firmware source code of a self-balancing robot.

Project developed for the robotics course at Galilean School of Higher Education (Padova, Italy).

- Microcontroller: PIC18F45K22
- Board: "Ready for PIC"
- IDE: MPLAB
- Compiler: XC8 (free)
- Libraries: PLIB
